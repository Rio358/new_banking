# Instalation

1) Add `new_banking` to your server.cfg
2) Drag the folder it into your resources folder
3) Modify as you wish

## Commands

> /bank Open Bank UI when near Bank | 
> /atm Open Bank UI when near ATM |
> /closebank Closes both UI if bugged
